# shiftlab-login

Реализованная форма для логина по заданию.

## Установка проекта

+ Склонировать репозиторий на локальную машину.
+ установить пакеты npm в папку репозитория `npm install`
+ Запустить приложение: `npm run dev`, перейти по адресу локального сервера

Запросы приходят по данному API: https://juniorsbootcamp.ru/api/otps
